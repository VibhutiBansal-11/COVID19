2. Laptop

## Setup RESTful API
To start the api , you need to install and setup flask and ngrok.

``` pip install flask```

[Install ngrok](https://ngrok.com/)

Clone this repository , navigate to the repository and run api.py

```python api.py```

and to start ngrok run

```ngrok http 5000```
